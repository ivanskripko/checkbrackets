Программа для проверки правильности выставления скобок в файле с кодом.
Работает через стек, т.е. каждый раз добавляем в стек открывающую скобку и вынимаем, когда у неё нашли пару.
Если стек в конце пуст, значит у каждой скобки есть пара, иначе в скобочном выражении ошибка.
В планах:
Учитывать скобки в комментариях и литералах
